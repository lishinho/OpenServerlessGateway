## 二、扩展能力规划

### 2.1 插件化扩展

基于 SPI 机制实现**全维度的插件化扩展**，支持第三方开发者自定义插件：

#### 2.1.1 协议扩展

| 扩展类型 | 说明 | 优先级 |
|----------|------|--------|
| 自定义协议解析器 | 支持企业内部私有协议 | 高 |
| Thrift 协议 | 支持 Apache Thrift | 中 |
| WebSocket 协议 | 支持实时通信场景 | 中 |
| MQTT 协议 | 支持 IoT 场景 | 低 |

#### 2.1.2 鉴权扩展

| 扩展类型 | 说明 | 优先级 |
|----------|------|--------|
| OAuth2.0 | 支持 OAuth2.0 标准协议 | 高 |
| SAML | 支持 SAML 单点登录 | 中 |
| LDAP | 支持 LDAP 企业目录 | 中 |
| OIDC | 支持 OpenID Connect | 中 |

#### 2.1.3 流量控制扩展

| 扩展类型 | 说明 | 优先级 |
|----------|------|--------|
| 自定义限流算法 | 支持自定义限流策略 | 高 |
| 智能熔断 | 基于机器学习的熔断 | 中 |
| 自适应限流 | 根据系统负载自适应 | 中 |

#### 2.1.4 日志监控扩展

| 扩展类型 | 说明 | 优先级 |
|----------|------|--------|
| 自定义日志插件 | 对接企业日志平台 | 高 |
| 自定义监控插件 | 对接企业监控平台 | 高 |
| 日志脱敏 | 敏感数据自动脱敏 | 中 |

#### 2.1.5 存储扩展

| 扩展类型 | 说明 | 优先级 |
|----------|------|--------|
| 自定义存储插件 | 对接企业存储系统 | 中 |
| 分布式缓存 | 支持多级缓存 | 中 |

## 三、版本规划详情

### 3.1 短期规划（V1.0 - V1.5）

#### V1.0 - 核心功能开源

**主要功能（核心能力）**：

| 功能模块 | 功能点 | 实现方案 | 状态 |
|---------|--------|---------|------|
| **限流** | 令牌桶算法 | 支持突发流量，平滑限流 | ✅ 已完成 |
| | 滑动窗口算法 | 精确限流，无突发流量 | ✅ 已完成 |
| | 多维度限流 | IP/接口/用户/业务/组合维度 | ✅ 已完成 |
| | 分布式限流 | Redis Lua 脚本实现 | ✅ 已完成 |
| **鉴权** | JWT 认证 | 无状态 Token，支持 RS256/HS256 | ✅ 已完成 |
| | RBAC 权限模型 | 基于角色的访问控制，支持角色继承 | ✅ 已完成 |
| | ABAC 权限模型 | 基于属性的访问控制，细粒度策略 | ✅ 已完成 |
| | 会话管理 | JWT无状态/Redis有状态/混合模式 | ✅ 已完成 |
| **HTTP转RPC** | 自动转发 | HTTP 请求自动转换为 RPC 调用 | ✅ 已完成 |
| | 泛化调用 | 无需依赖服务接口 SDK 即可调用 | ✅ 已完成 |
| | 多协议支持 | Dubbo/gRPC/SOFA 协议适配 | ✅ 已完成 |
| | 服务发现 | Nacos/Zookeeper/Consul 集成 | ✅ 已完成 |
| **灰度发布** | 多维度灰度 | 用户/IP/Header/权重/业务属性 | ✅ 已完成 |
| | 规则配置 | 动态配置，实时生效 | ✅ 已完成 |
| | 自动回滚 | 基于监控指标的自动回滚 | ✅ 已完成 |

**部署支持**：
- [x] Docker 镜像发布
- [x] K8s 部署文件
- [x] 传统虚拟机部署

**框架适配**：
- [x] Spring Cloud 适配
- [x] Dubbo 适配
